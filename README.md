## Requirements

- PHP >= 8.1
- Laravel 11.x
- Composer
- MySQL or other database


## Установка

Пошаговая инструкция по установке проекта.
 * composer update
 * php artisan key:generate
 * php artisan migrate
 * npm i
 * npm run dev
